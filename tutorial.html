IS.bookmarks.html is the websites database of Illegal Services.
The goal is to update it as much as possible to gather a maximum number of links.
This file is updated at least once a week to always stay updated.

Illegal Services Windows app will download this file and parse it using bookmarks_parser.exe to display it in a human-readable format.
But it is WAY easier and comfortable to use this file into your web browser bookmarks, as their original bookmarks format.

How to import IS.bookmarks.html into your bookmarks navigation bar?
Well, according to your web browser and your system, choose a link from the bottom
and follow their instructions, it's really easy!
One last important thing, you need to remember that the bookmarks database file
is 'IS.bookmarks.html' and it is available a little bit everywhere in this website, but also here:
https://raw.githubusercontent.com/Illegal-Services/Illegal_Services/downloads/IS.bookmarks.html
You will need to save it, I recommend the good old CTRL+S once you opened the link.
